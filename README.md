# FreakyBrowse!
This is a browser completely written in Python! It has many features, such as:
- Notes
- Downloads
- Themes

and many more!
<hr>
To run FreakyBrowse, just run the command `py freakybrowse.py` inside the directory where freakybrowse.py is located.
